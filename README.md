# Task Tracker (CRUD)

A simple full-stack task tracker application built to refresh and demonstrate core CRUD operations.

## Features
- Create tasks
- View all tasks
- Edit existing tasks
- Delete tasks

## Tech Stack
- Backend: Node.js, Express, MongoDB, Mongoose
- Frontend: HTML, Vanilla JavaScript, Bootstrap
- Tools: Nodemon, Git, GitHub
